# ICO Converter JS

This project is a rewrite of my Python fastHTML app, rebuilt so the conversion logic runs on the client instead of the server.

It lets you upload an image in the browser, converts it to `.ico`, and downloads the result without sending files anywhere.

## What it does

- Accepts common image formats like PNG, JPG, JPEG, WebP, and BMP
- Builds ICO sizes in the browser
- Shows a preview before download
- Keeps the entire workflow client-side
